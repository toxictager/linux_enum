# Linux Enum Tool 🐧

A basic Linux system enumeration script written in Python.

This script collects:

- System info (kernel, architecture, OS)
- Logged-in users
- System uptime
- Disk usage
- Network interfaces
- Open ports

📁 Output is saved to: `output/report.txt`

## 💻 Usage

```bash
python3 linux_enum.py
```
